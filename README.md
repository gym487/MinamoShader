
##Minamo Shader

A simple shader based on path-tracing.

Edit src/minamo.java to change the scene and settings.

Run:

	javac  ./src/* -d ./bin/
	java -cp ./bin/ minamo

First time to run:

	mkdir ./pictures
	mkdir ./bin
